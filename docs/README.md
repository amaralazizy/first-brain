# Monorepo Documentation

This directory contains all docs-as-code for the First Brain project.

- `architecture/` — System diagrams, stack choices, and high-level overviews
- `adr/` — Architecture Decision Records (ADR-001, ADR-002, ...)
- `runbooks/` — Setup, local dev, deployment, troubleshooting
- `decisions.md` — Running log of major technical decisions
- `stack.md` — Registry of all major dependencies, status, and rationale

> All major changes must be accompanied by an ADR or decision log entry.
